# API Endpoints

## Users
- POST /api/users/register
- POST /api/users/login
- GET /api/users/profile
- PATCH /api/users/profile

## Items / Resources (Edit based on your project)
- POST /api/items
- GET /api/items
- GET /api/items/:id
- PATCH /api/items/:id
- DELETE /api/items/:id

## Admin
- GET /api/admin/users
- DELETE /api/admin/users/:id
